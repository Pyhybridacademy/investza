VAPID keys not configured — push will not work

Chrome and most browsers require VAPID keys to allow push subscriptions. Without them, the browser will silently reject the subscription and 0 subscribers will be recorded. Follow the steps below to fix this.

# 1. Install pywebpush on your server
pip install pywebpush

# 2. Generate VAPID keys
vapid --gen

# 3. Add to your Render environment variables:
VAPID_PRIVATE_KEY = <private_key from output>
VAPID_PUBLIC_KEY = <public_key from output>

# 4. Add pywebpush to requirements.txt
echo "pywebpush" >> requirements.txt

After adding the env vars, redeploy on Render. Users who previously allowed notifications will need to revisit the site once for their subscription to register.
